# 肠胃脆弱单页 · 设计（小白简版 / 数据详版）

> **是什么**：特殊人群食品评分中「肠胃脆弱」单人深用版，独立成页，用来打磨"按画像分层 + 直接可照做"的体验。
> **在线**：https://shi722haha.github.io/everly-gut-solo/ ｜ 源码：https://github.com/shi722haha/everly-gut-solo
> **关系**：六类人群全量版见 `everly-people-scores`（不动）；本页是肠胃这一类的深做与对比。
> **用户原型**：40 岁 · 常出差 · 消化弱 · 9:00 喝咖啡（沿用肠胃飞轮虚拟用户）。
> **状态**：可点击原型 + 设计记录。wellness 健康管理，不做医疗诊断。

---

## 0. 一句话定位

同一套肠胃评分，**生成 mini app 时可选两种展示**——给 50% 小白的「小白简版」（零指标、直接照做）和给 25% bio hacker 的「数据详版」（分数 + 区间带 + 数据）。**少数据是一个可选参数，不是写死的界面。**

验收标准（大鹏）：**他不爱看数据，瞄一眼就能直接照做。**

---

## 1. 模式开关（顶部）

`😀 小白简版 ｜ 📊 数据详版`，默认小白简版，状态记到 localStorage（key 带 `_solo`，与全量版互不干扰）。

- **小白简版**：零指标、按画像表层，下文 §2。
- **数据详版**：原全数据版——分数环（0–100）、宏量、区间带因素行（带 ± 分、出处可展开）、AI 洞察分析。给想看数据的人。

## 2. 小白简版 · 模块顺序与职责

| # | 模块 | 职责 / 要点 |
|---|---|---|
| 1 | **整体判定** | 「今天肠胃 · 要当心」+ 一句大白话。零数字，给整体感受。 |
| 2 | **🍽 今天吃了什么** | 带真实菜品图的当天餐食记录（日常记录不能少）。 |
| 3 | **🍙 脂肪 / 碳水 / 蛋白 + 按年龄分析** | 三条进度条 + **「按你这个年龄（40 岁）该怎么吃」**：蛋白要吃够（防肌肉流失）、碳水别超量、脂肪护血管。说明"年龄不同需求不同"。紧跟饮食记录。 |
| 4 | **✨ AI 洞察** | 总结句常显（出差/作息/压力也在拖后腿，不只食物）；下面分析（确认诱因/观察中/可安心三档）折叠可查。**含一行意图识别小字**（见 §3）。 |
| 5 | **🎯 给自己定个小目标** | 进度条 +「已稳 3/7 天，还差 4 天达成」常显；换目标 / 自己写**折叠**。给他一个冲刺空间。 |
| 6 | **🚫 今天这几样别再吃了** | 点名今天吃的具体食物（油条/麻辣香锅/豆浆/咸菜）+ **归因**（为啥 + 追到哪顿）+ **三档反馈**（见 §3）。 |
| 7 | **🔍 帮你找雷** | 给"不知道自己啥不能吃"的人：列正在观察的几样（豆制品/辛辣/乳制品/生冷/咖啡），吃完反馈几次就帮他认准；可"＋我怀疑还有别的"。 |
| 8 | **🥗 今天该少吃 / 该补的** | 营养成分行**去专业化**：表面只露**行动指令 + 普通食材名**，「多/少」条码折进去可查（见 §3）。 |
| 9 | **🍳 换着吃菜谱** | 留首道常显，其余折叠（▾ 再看 N 道）。 |

## 3. 几条核心设计决策（这页的灵魂）

### 3.1 表述日常化 + 去专业化
不写「高 FODMAP / 高油 / 高盐」这种黑话；直接说**今天哪样食物、为什么**：
- 「高 FODMAP」→「爱胀气的（豆浆 / 洋葱 / 蒜）」
- 「脂肪·高脂油炸」→「油大的（油炸 / 肥肉）」；「钠」→「太咸的（咸菜 / 腌的）」

### 3.2 行动指令，不只是标签（修改意识摆上台面）
每行先给**指令**：今天吃过的 →「少吃 / 先别吃」，没吃够的 →「该补 / 加点」，并说清"今天那杯豆浆空腹喝多了，明天少来点"。

### 3.3 直观在表面，详情折叠可查
食材名一眼就懂；那条**「少—刚好—多」区间带**折叠在里面，想查再点开。AI 洞察同理：总结句常显、分析折叠。

### 3.4 用户矫正 = 三档，用户说了算（不定死）
"别吃"的每样问：**吃完之后肠胃什么感觉？😌 挺舒服 ｜ 😐 没感觉 ｜ 😣 有点难受**。
- 「没感觉」单列——没反应也是一种状态（=没症状，基线）。对齐 GutDiaries / myIBS / Bloom 的「无→严重」量表与"吃完约 2 小时记反应"。
- 舒服/没感觉 → 以后少提醒；难受 → 帮你盯紧。越选越懂他。

### 3.5 意图识别：跨 app 行为折射，轻提即可
不单独做卡片。意图是从**他愿不愿采纳运动/睡眠建议、坚不坚持记录**里折射出来的"想改变的人"。只在 AI 洞察里一行小字提一句，识别到"想改"就把方案给得更主动。

### 3.6 影响因素不只食物
肠胃的雷还来自**空腹咖啡、出差/作息打乱、压力、睡眠、睡前进食、两餐间隔太长**。用「确认诱因 / 观察中 / 可安心」三档承接（多信号交叉验证 + 四层记忆）。

### 3.7 年龄段独立分析
脂肪/碳水/蛋白人人都有，但**每个年龄需求不同**——年轻人能多碳水，上年纪要多蛋白、控脂肪。按用户年龄（这里 40 岁）给一条独立分析。
