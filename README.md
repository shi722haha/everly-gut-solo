# Everly · 肠胃脆弱 单页（小白简版 / 数据详版）

特殊人群食品评分——肠胃脆弱单人深用版，独立成页便于对比与迭代。

**在线体验：https://shi722haha.github.io/everly-gut-solo/**

- 顶部开关「😀 小白简版 / 📊 数据详版」= 生成 mini app 时可选的参数（不是写死界面）
- 小白简版：零指标，直接“吃啥/别吃啥”，食物具体；归因（追到今天哪顿）+ 三档反馈（挺舒服/没感觉/有点难受，你说了算）+ 大白话
- 模块：整体判定 · 今天吃了什么 · 脂肪/碳水/蛋白+按年龄分析 · AI 洞察（含意图识别小字）· 给自己定个小目标 · 别吃 · 帮你找雷 · 该少吃/该补 · 换着吃菜谱
- 数据详版：分数环 + 区间带 + 宏量 + 因素行（给 bio hacker 看数据）

> wellness 健康管理，不做医疗诊断。配套设计见 everly-people-scores 仓库。
